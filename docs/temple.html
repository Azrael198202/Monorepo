大明寺（だいみょうじ｜Daming Temple）

大明寺は唐代に創建された古刹で、長い歴史の中で幾度も修復と再建を重ねてきました。
現在も揚州を代表する仏教寺院として、多くの人々に親しまれています。

空間建築

山門から殿堂へ：段階的に深まる静けさ

  • 軸線：中軸対称の構成で、秩序と儀礼性を際立たせます。
  • 庭：歩みとともに景が移ろい、視線は開放から収束へ向かいます。

アクセス方法

  • 市内より：車で約XX分
  • 公共交通：XX線 → XX駅 → 徒歩
  • アドバイス：ラッシュ時を避け、待ち時間に余裕を持って

参拝ルート

  • 山門 → 本堂 → 回廊 → 庭園
  • 静粛/撮影ルール：現場の案内に従ってください
  • オプション：写経/香礼（要予約）

注意事項

  • 服装：過度に肌を露出しない服装で
  • 礼儀：法器/供物に触れない
  • 天候：雨具/日焼け対策/歩きやすい靴
